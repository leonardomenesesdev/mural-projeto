Vivere+


Descrição

Vivere+ foi um projeto desenvolvido no Ciclo 1 e remete a experiência de um e-commerce para criação, gestão e compartilhamento de experiências personalizadas (trilhas, yoga, workshops), indo além do modelo tradicional de ingressos de eventos.
A aplicação inclui funcionalidades essenciais como autenticação de usuários, carrinho de compras, e operações de CRUD (criação, leitura, atualização e remoção) integradas a um banco de dados. Com uma arquitetura completa de frontend e backend, o projeto simula um fluxo típico de uma loja virtual, promovendo a compreensão prática de conceitos como rotas, persistência de dados e manipulação de estados.

BookFlow


Descrição

BookFlow também foi um projeto desenvolvido no Ciclo 1, o qual buscou trazer uma visão mais administrativa de um e-commerce de livros. A aplicação também inclui autenticação de usuários, carrinho de compras, e operações de CRUD (criação, leitura, atualização e remoção) integradas a um banco de dados. Com uma arquitetura completa de frontend e backend, o projeto simula boa parte de um fluxo típico de um administrador de loja virtual, promovendo a compreensão prática de conceitos como rotas, persistência de dados e manipulação de estados.
